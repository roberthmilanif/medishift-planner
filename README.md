# MediShift Planner - Passômetro UTI

Sistema otimizado de passagem de plantão para UTI.

## Funcionalidades

✅ **DIH Automático** - Cálculo automático de dias de internação  
✅ **Antibioticoterapia Estruturada** - Contagem de dias (Ex: Meropenem D7)  
✅ **Busca e Filtros** - Por leito, prioridade e autor  
✅ **Sincronização Automática** - Atualizações em tempo real entre plantonistas  
✅ **Interface Moderna** - Design limpo e responsivo
